Ajuste de precios
Redeterminación
Ajuste por variación de costos.
Tiene Porcentaje, Salto (mes de inicio) y Nro (orden secuencial). Su monto se calcula, no se ingresa manualmente.
🔄
Pago derivado
Reliquidación
Pago generado automáticamente para cada certificado emitido después del Salto de una redeterminación. El primer certificado post-Salto absorbe retroactivamente el porcentaje ya ejecutado pre-Salto (ver Sección 4, Paso 3b). Representa la parte proporcional del ajuste.

3. Fórmula del MontoRedet (CalcularMontoRedeterminaciones)

Las redeterminaciones se procesan en orden por Nro (1, 2, 3…). Solo se consideran certificados emitidos antes del Salto para determinar el porcentaje ya utilizado.
Paso A — Porcentaje de ejecución

Sumamos los certificados del autorizante cuya MesAprobacion es menor al Salto de la redeterminación.

SumaCerts = cert.MontoTotal donde cert.MesAprobacion < redet.Salto PorcentajeUtilizado = (SumaCerts / auth.MontoAutorizado) × 100
Paso B — Faltante

El faltante representa qué porcentaje del autorizante aún no fue pagado al momento del Salto.

Faltante = Max(0, 100 - PorcentajeUtilizado) // Nunca negativo: si ya se pagó el 110%, el faltante es 0
Paso C — MontoRedet para Nro = 1 (primera redeterminación)
MontoRedet[Nro=1] = ( auth.MontoAutorizado × redet.Porcentaje / 100 ) × ( Faltante / 100 )
📌 redet.Porcentaje -> Porcentaje de ponderacion
Paso C — MontoRedet para Nro > 1 (redeterminaciones posteriores)

Las redeterminaciones anteriores del mismo autorizante se suman al monto base, acumulando el ajuste sobre la base ampliada.

MontoRedet[Nro>1] = ( (auth.MontoAutorizado + SumaRedetsAnteriores) × redet.Porcentaje / 100 ) × ( Faltante / 100 )
📌 Ejemplo con R2: MontoAutorizado=$1.000.000, R1.MontoRedet=$50.000, Porcentaje R2=8%, Faltante=30%
→ Base = 1.000.000 + 50.000 = $1.050.000
→ MontoRedet R2 = (1.050.000 × 8/100) × (30/100) = 84.000 × 0,3 = $25.200

4.2 Fórmulas paso a paso

Paso 1 — Porcentaje del certificado respecto al autorizante

Indica qué fracción del MontoAutorizado total representa ese certificado.

cert.Porcentaje = (cert.MontoTotal / auth.MontoAutorizado) × 100 // Ej: $250.000 / $1.000.000 × 100 = 25%
Paso 2 — Condición para generar la reliquidación

Se evalúa para cada combinación de redeterminación y certificado del mismo autorizante.

SI cert.CodigoAutorizante == redet.CodigoAutorizante Y cert.MesAprobacion > redet.Salto // Solo certificados POSTERIORES al Salto reciben reliquidación // El PRIMERO de ellos absorbe retroactivamente el porcentaje acumulado pre-Salto (ver Paso 3b)
Paso 3a — Cálculo del monto (certificados posteriores al Salto)

Para todos los certificados con MesAprobacion > redet.Salto.

MontoReliq = redet.MontoRedet × ( cert.Porcentaje / 100 )
// Expandido (sustituyendo cert.Porcentaje): MontoReliq = redet.MontoRedet × ( cert.MontoTotal / auth.MontoAutorizado )
// Expandido total para Nro=1 (sustituyendo MontoRedet): MontoReliq = cert.MontoTotal × (redet.Porcentaje/100) × (Faltante/100)
📌 Ejemplo: MontoRedet=$50.000 · cert.MontoTotal=$250.000 · MontoAutorizado=$1.000.000
→ cert.Porcentaje = 250.000/1.000.000 = 25%
→ MontoReliq = $50.000 × 25% = $12.500

Expandido total: $250.000 × 10% × 50% = $250.000 × 0,05 = $12.500 ✓
Paso 3b — Cálculo del monto (primer certificado post-Salto)

Para el primer certificado cuyo MesAprobacion > redet.Salto, el porcentaje acumulado pre-Salto se suma al porcentaje propio: este primer pago absorbe retroactivamente el trabajo ya ejecutado antes del Salto.

// Variables previas porcentajeAcumulado = (SumaCerts_preSalto / auth.MontoAutorizado) × 100 certPorcentaje = (cert.MontoTotal / auth.MontoAutorizado) × 100
// Fórmula para el primer cert post-Salto MontoReliq = redet.MontoRedet × ( (certPorcentaje + porcentajeAcumulado) / 100 )
📌 Ejemplo: Salto = Jun-2024, primer cert post-Salto = Cert 3 (Ago-2024) por $250.000.
→ certPorcentaje = 250.000 / 1.000.000 = 25%
→ porcentajeAcumulado (Cert1 + Cert2 pre-Salto) = 500.000 / 1.000.000 = 50%
→ MontoReliq = $50.000 × (25% + 50%) = $50.000 × 75% = $37.500

Con la fórmula estándar (Paso 3a) habría dado: $50.000 × 25% = $12.500
✅ Regla de consistencia: El total de todas las reliquidaciones de una redeterminación es igual a MontoRedet × ( cert.Porcentaje post-Salto / 100). Si los certificados futuros cubren exactamente el porcentaje faltante, el total reliquidado iguala al MontoRedet.
